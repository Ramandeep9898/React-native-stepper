## Run the app locally

### Clone the project

```bash
  git clone https://github.com/betterhalf-ai/wedding-web-app.git
```

### Go to the project directory

```bash
  cd React-native-stepper
```

### Install dependencies

```bash
  yarn
```

### Run the Development Server

```bash
  yarn dev
```
